# Dinumbers

Dinumbers is a small Angular quiz app for preschool number-symbol practice. Each round shows zero to nine colorful vegetables and asks the player to choose the matching digit.

## Game Rules

- Starts with 3 answer choices.
- Every 3 correct answers adds one more answer choice.
- After 3 wrong tries, the game ends and displays the number of correct answers.
- Difficulty and retry behavior are configured in `src/app/app.ts`.

## Scripts

```bash
npm start
npm run build
npm run lint
npm test -- --watch=false
npm run format
```

The development server runs at `http://localhost:4200/` by default.
